 Playwright Login Automation

 Overview
This project demonstrates UI automation testing using Playwright for a login workflow.

 Tech Stack
- JavaScript
- Playwright

 Test Scenarios
- Valid login
- Invalid login

 Setup & Run

1. Install dependencies:
npm install

2. Install browsers:
npx playwright install

3. Run tests:
npx playwright test

 Key Features
- Automated UI testing
- Assertions for validation
- Structured test cases

 Purpose
This project showcases hands-on experience with Playwright automation framework, including element handling and validation.
